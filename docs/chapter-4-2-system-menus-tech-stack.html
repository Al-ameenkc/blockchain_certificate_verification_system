4.2 System Menus Implementation

The administrative frontend is organised around a persistent sidebar menu (Home, Records, Issued, Registered) plus Logout, routing to /dashboard, /dashboard/record, /dashboard/issued, and /dashboard/registered respectively. The Home view implements the dual workflows issue and register legacy certificate preparation before navigation to /certificate/[ref] for preview and blockchain submission. Public verification is exposed at /verify and /verify/[ref] without the sidebar. Entry and session boundaries use /login, Next.js middleware, and a server-issued session cookie for protected routes.
